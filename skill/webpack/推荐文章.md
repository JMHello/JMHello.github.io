8、missing class properties
在设置了 modules:false后就出现 missing class properties

安装 babel-plugin-transform-class-properties
并在.babelrc的plugins中配置

```json
{
  "presets": [
    "react",
    "es2015",
    "stage-1",
    "stage-2",
    "stage-3",
    [
      "env",
      {
        "modules": false
      }
    ]
  ],
  "plugins": [
    ["import", {
      "libraryName": "antd",
      "style": "css"
    }],
    "transform-runtime",
    "transform-class-properties",
    "syntax-dynamic-import"
  ]
}

```
