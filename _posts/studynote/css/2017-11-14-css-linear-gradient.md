### 2.1 语法

```
background/background-image: linear-gradient([<angle> | to <side-or-corner>], color-shop, ..., color-shop);

background/background-image: repeating-linear-gradient([<angle> | to <side-or-corner>], color-shop, ..., color-shop)
```
